# ¿Alguien pidió un server de literatura activo y divertido?

**¡Bienvenido a Los escritores olvidados!**

Una pequeña comunidad donde podrás mostrar tu trabajo, recibir retroalimentación y charlar con otros escritores. El respeto es primordial y pasarle bien es regla. Contamos con expertos en la materia y diversas dinámicas literarias a las que sacarles provecho. Únete a la tertulia de esta biblioteca olvidada de la mano de Dios.
